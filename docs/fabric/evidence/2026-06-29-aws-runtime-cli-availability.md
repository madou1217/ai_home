## Local Code Fix Found During Readback

The first real local readback exposed a CLI output bug:

```text
fabric nodes aws-current-node --json
stdoutBytes=8192
JSON.parse failed: Unterminated string in JSON at position 8192
```

Root cause: `fabric-router` wrote a large JSON payload and immediately called
`process.exit()`, so piped stdout could be truncated before flush.

Fix:

- `lib/cli/commands/fabric-router.js` now waits for real `process.stdout`
  write completion before exiting command branches.
- Fake test stdout objects still resolve synchronously.

Verification after the fix:

```text
fabric nodes aws-current-node --json
stdoutBytes=14070
JSON.parse=ok
```

The same `fabric-router.js` fix was copied to AWS current and checked there:

```text

```

# AIH Desktop UI 高保真线框图与产品交互设计稿

本稿件基于 **Ant Design Pro (UmiJS) & Tauri 桌面端混合应用框架**，针对原有系统松散、平铺、粗糙的程序员 UI 进行了彻底的结构重构和交互重塑。

---

* **产品思路**：
  * **页头Descriptions集成**：将原本平铺在外的 10 个丑陋统计卡片彻底移除，将可用率、待处理数、停用数等状态信息作为 `PageScaffold` 的 `Descriptions` 内嵌在页面副标题右侧，清爽护眼。
  * **Badge 状态灯**：使用 `● 已配置` / `● 未配置` 精致的 Badge 状态灯（非生硬色块 Tag）表示配置，并引入 ProTable 自带的 `Pagination` 高级圆角分页，极大提升档次。
  * **动作区上提**：添加、导入、导出操作统一在 `PageScaffold.extra`（右上角）聚合展示。
